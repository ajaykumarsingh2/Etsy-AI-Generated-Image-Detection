# Etsy-AI-Generated-Image-Detection

Title: AI-Generated Image Detection for Etsy Marketplace
Overview: This project addresses the "Trust Crisis" on Etsy by building a Deep Learning classifier to identify synthetic images generated by DALL-E/Midjourney.
Tech Stack: Python, PyTorch, MobileNetV2, Transfer Learning.
Key Results:
Optimized model with 224x224 resolution to capture micro-textural artifacts.
Achieved an F1-score of 0.7719.
Performed Threshold Grid-Search (0.65) to minimize False Positives.
